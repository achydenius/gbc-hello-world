# Game Boy Color Hello World

- Install [RGBDS](https://rgbds.gbdev.io/) (`brew install rgbs` works on MacOS)
- Compile with `make`
- Run *main.gb* with e.g. [Sameboy](https://sameboy.github.io/) or a real device
